# Cryptocurrency Tracker

A beautiful single-page application built with pure Vanilla JavaScript.

---

## Features

### Dashboard
- Total Market Cap
- 24h Trading Volume
- BTC Dominance
- Active Coins & Markets
- Fear & Greed Index
- Market Trend

### Coins Table
- Full sorting by any column
- Instant search by name/symbol
- Responsive & mobile-friendly

### Coin Detail Page
- Current price & 24h change
- 7-day price chart 
- All-Time High/Low
- Coin description

## Tech Stack
- **Vanilla JavaScript** 
- **Tailwind CSS** 
- **Handlebars**  
- **Chart.js** 
- **CoinGecko API**
- **Alternative.me API** 


## Local Development

```bash
git clone https://github.com/fioneo/crypto-tracker.git
cd crypto-tracker
npm install
npm run dev
```
